# Support チーム サンプル依頼集

`/support` 起動後に使える依頼例です。

---

## 運用監視部

### 障害対応
```
夜間バッチが異常終了しました。
ジョブ名: 売上集計バッチ
終了コード: 1
ログに「ORA-01555: snapshot too old」が出ています。
影響範囲と調査手順を教えてください。
```

```
本番Webサーバへのアクセスが急増し、レスポンスが遅くなっています。
CPU使用率が90%を超えています。
まず何を確認すればいいですか？
```

### 定期作業の改善
```
毎月月初に手動でデータ移行作業をしています。
作業時間が2時間かかっていて属人化しています。
自動化・標準化の提案をしてください。
```
