# UserSupport チーム サンプル依頼集

`/support` 起動後、社内ユーザー対応として使える例です。

---

## PC・端末サポート部

```
社員のPCが起動しなくなったと連絡がありました。
電源ボタンを押しても画面が真っ暗なままです。
ユーザーへの確認手順と対応手順を教えてください。
```

```
プリンターが突然使えなくなりました。
「ドライバーが利用できません」というエラーが出ています。
対処手順を教えてください。
```

```
PCの動作が非常に遅くなっています。
ユーザーから「最近Windows Updateがあった」と聞いています。
確認・対処の手順を教えてください。
```

---

## アプリ操作サポート部

```
社員から「Outlookでメールが送信できない」という問い合わせがありました。
エラーメッセージ: 「0x800CCC0F 接続が切断されました」
対処手順をユーザーに案内する文面で教えてください。
```

```
Excelのファイルが開けないという問い合わせがありました。
「ファイルが壊れています」というメッセージが出ています。
確認手順と復旧方法を教えてください。
```

---

## アカウント・権限管理部

```
新入社員のアカウント作成依頼が来ました。
必要な申請情報のチェックリストと作成手順を教えてください。
```

```
退職者のアカウント無効化が必要です。
対応すべき項目のチェックリストを作ってください。
（AD / メール / 業務システム / VPN 等）
```

```
社員から「共有フォルダにアクセスできない」という問い合わせがありました。
権限確認の手順と申請フローを案内してください。
```

---

## ネットワーク・接続サポート部

```
在宅勤務中の社員からVPNが繋がらないという連絡がありました。
エラー: 「The VPN connection failed due to unsuccessful domain name resolution」
対処手順をユーザーに案内する形で教えてください。
```

```
社員のPCがWi-Fiに繋がらなくなりました。
有線LANでは繋がるとのことです。
確認・対処の手順を教えてください。
```
